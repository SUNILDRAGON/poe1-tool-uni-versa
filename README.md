# POE1 Legacy Toolkit – Offline Companion for Path of Exile 1

**Open-source, standalone tools for planning, analysis, and reference — no game interaction.**

---

## Introduction

**POE1 Legacy Toolkit** is a collection of **offline utilities** for Path of Exile 1 players who want to plan builds, study mechanics, and keep legacy data organized — all outside the game client.

- Works **completely offline**  
- **No overlays, no input simulation, no screen reading**  
- **No connection to the game process**  
- Pure reference, planning, and data tools  
- MIT-licensed, fully auditable on GitHub  

> *"Like a digital notebook for the old Wraeclast."* — Legacy enthusiasts

---

## Core Features (All Offline & Non-Invasive)

| Feature | Description |
|--------|-----------|
| **Passive Tree Planner** | Full interactive skill tree with import/export (PoB-compatible). Search nodes, simulate links, export codes. |
| **Item Base Database** | Searchable offline index of all PoE1 bases, mods, and tags. Filter by type, ilvl, influence. |
| **Crafting Bench Reference** | Complete list of bench crafts, costs, and unlock requirements. Filter by mod group. |
| **League Mechanic Archive** | Documented summaries of every league mechanic (Blight, Delirium, etc.) with maps, rewards, and strategies. |
| **Build Gallery** | Community-submitted legacy builds with tree, gear, and notes. Browse, fork, export. |
| **Custom Filter Generator** | Create and edit loot filters locally. Syntax highlighting, live preview, export to `.filter` files. |
| **Dark Theme UI** | Clean, minimal interface with retro PoE1-inspired fonts and adjustable scaling. |

*All tools run independently. No runtime game attachment.*

## Why Use It?

Path of Exile 1 is deep, complex, and full of legacy knowledge that’s hard to access quickly. This toolkit keeps everything in one place — **without touching the game**.

- Plan your next character in minutes  
- Look up mods without Alt+Tabbing to the wiki  
- Preserve strategies from leagues long past  
- Generate filters with confidence
